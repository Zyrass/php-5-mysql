Les formulaires

Chaque formulaire commence par une balise form avec comme attribut :

  • name qui correspond au nom du formulaire
  • action qui stipule la page qu'on aura une fois la soumision faîtes.
  • method qui permet de définir si on souhaite un envoi via l'url ou autrement.
        <form name="form1" action="index.html" method="post">
            ... le formulaire ici ...
        </form>

A savoir que si par défaut on ne stipule pas d'action alors on chargera la même page.
Également si on ne stipule pas de méthode, alors par défaut nous aurons la méthode GET qui sera sélectionner.
Par mesure de sécurité on utilise pratiquement tout le temps la méthode POST

Fieldset et sa légende pour donner du style à nos formulaires.

        <fieldset>
            ... Un partie du formulaire ...
        </fieldset>
Exemple :
Qui êtes-vous ?
Vos 2 pseudos préféré

Plusieurs type sont possible :

  • text permet d'inscrir du texte
  • button permet de concevoir un bouton
  • image permet d'ajouter une image avec l'attribut src
  • password pour les mots de passe
  • checkbox Permet de sélectionner des cases
  • radio Permet de choisir 1 résultat sur une certaine liste
  • hidden Permet de définir un champ caché
  • submit Permet d'envoyer le formulaire
  • reset Permet d'annuler le formulaire
  • email Pour les email
  • url Pour les adresse web
  • file Pour charger un fichier local
  • textarea Permet de créer un champ multilignes
  • select Permet de gérer une liste déroulante. Accompagné des options avec sa propre value
